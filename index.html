How to create a folder?

Copy

New-Item -ItemType Directory -Path d:/new_folder_using_powershell

How this code works?

  • You are making use of New-Item cmdlet to create a new item (in our case, it's a directory or folder)
  • Then pass in the item type value (Directory) to -Itemtype parameter
  • Finally specify the path (using -Path parameter) where you want the directory to be created. (In case you are already in the location where you want to create a folder, then you can execute the New-Item cmdlet without the -Path parameter as shown in the gif).

    • How to create a file?

    Copy

    New-Item File1.txt

    How this code works?

  • You are making use of New-Item cmdlet to create a new item (in our case, it's a file).
  • Then pass in the name of the file (example: file.txt)
  • (You can also pass in the item type to the -ItemType parameter like this: -ItemType "file", if necessary.

    • How to add multiple files?

    Copy

    (0..10) | ForEach-Object {New-Item File$_.txt}

    How this code works?

  • You are making use of ForEach-Object cmdlet and running it for the limit: 0..10.
  • For every iteration, you create a new file using New-Item cmdlet.
  • Finally, you pass in the iterator ($_.) as a part of the file name: File$_.txt to distinguish between the files being created.

    • How to write into a file?

    Copy

    Add-Content File2.txt -Value "hello world"

    How this code works?

  • You are making use of Add-Content and pass in the file name to it (for which you are going to add the content).
  • Then pass in the content itself to the -Value parameter (example: "Hello World").

    • How to open a file?

    Copy

    Invoke-item File1.txt

    How this code works?

  • You pass in the name of the file (which you wish to open) to the Invoke-item cmdlet
  • The file opens up when you execute the cmdlet

    • How to get a file's content?

    Copy

    Get-Content File1.txt

    How this code works?

  • You pass in the name of the file (which you wish to read) to the Get-Content cmdlet
  • The contents of the file get displayed when you execute the cmdlet


    • How to overwrite an existing file?

    Copy

    New-Item File1.txt -Force

    How this code works?

  • You pass in the Force parameter as you create the new file using the New-Item cmdlet
  • Doing so will overwrite the existing file with the new one you just created.


    • How to delete a file?

    Copy

    Remove-Item File1.txt

    How this code works?

  • You pass in the file name to the Remove-Item cmdlet


    • How to delete multiple files of same type?

    Copy

    Remove-Item *.txt

    How this code works?

  • You pass in the PowerShell wildcard * (which represents everything) along with the file extension to the Remove-Itemcmdlet.
  • The above command removes all the files with a .txt extension available within the folder_usig_powershell folder.


    • How to delete a folder?

    Copy

    Remove-Item [folder_name]

    How this code works?

  • You pass in the folder name to the Remove-Item cmdlet
